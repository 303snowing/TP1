## 《Linux Basics for Hackers》

- 书籍：《Linux Basics for Hackers》2019版
- 目的：暂时主要为翻译团队内部使用
- 时间：预计2019/01/10--2019/04/24
- 人员：预计5人左右，共同承担全部章节翻译任务
